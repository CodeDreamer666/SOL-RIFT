# SOL // RIFT

SOL // RIFT is a self-contained 3D endless-action runner built with TypeScript,
Three.js and Vite. Every run uses a new seeded course assembled from validated
challenge patterns, five environment styles, recovery sectors and escalating
multi-action combinations.

## Run locally

```bash
npm install
npm run dev
```

Open the local URL printed by Vite.

## Controls

| Action | Keyboard |
| --- | --- |
| Steer / hold into a wall-run surface | `A` / `D` or left / right arrows |
| Jump / wall kick | `Space`, `W` or up arrow |
| Slide | `S` or down arrow |
| Dodge left / right | `Q` / `E` |
| Dash through closing gates | `Shift` |
| Pause / resume | `Esc` or `P` |
| Restart after a run | `R` |
| Mute | `M` |

Touch and pointer action controls are included for narrow screens.

## Quality checks

```bash
npm run lint
npm run typecheck
npm run test
npm run build
```

`npm run check` runs the complete suite.

Run history, display name and sound/motion preferences stay in the browser's
local storage. The scoreboard is intentionally local-only.
